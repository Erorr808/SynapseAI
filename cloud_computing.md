# Cloud Computing and Infrastructure

Cloud computing delivers computing services over the internet, providing scalable, on-demand resources without direct management.

## Cloud Computing Models

### Infrastructure as a Service (IaaS)
- **Virtual Machines**: Complete server virtualization
- **Storage Systems**: Scalable data storage solutions
- **Networking**: Virtual network configuration
- **Load Balancers**: Traffic distribution management
- **Content Delivery Networks**: Global content distribution

### Platform as a Service (PaaS)
- **Development Frameworks**: Pre-configured development environments
- **Database Services**: Managed database instances
- **Application Runtimes**: Language and framework support
- **Middleware**: Application integration services
- **DevOps Tools**: CI/CD pipeline support

### Software as a Service (SaaS)
- **Productivity Applications**: Office suites, collaboration tools
- **Business Applications**: CRM, ERP, HR systems
- **Communication Tools**: Email, messaging, video conferencing
- **Analytics Platforms**: Business intelligence tools
- **Creative Software**: Design and media applications

### Function as a Service (FaaS)
- **Serverless Computing**: Event-driven execution
- **Microservices**: Small, independent functions
- **Event Processing**: Trigger-based computation
- **API Gateways**: Function orchestration
- **Auto-scaling**: Demand-based resource allocation

## Cloud Service Providers

- **EC2**: Elastic Compute Cloud virtual servers

### Google Cloud Platform (GCP)
- **Compute Engine**: Virtual machine instances
- **Cloud Storage**: Unified object storage

- **Cloud SQL**: Managed database service
- **Deployment Manager**: Infrastructure automation
